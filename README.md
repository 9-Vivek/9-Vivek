# Developer Portfolio & Research Hub

An Astro and Tailwind portfolio for Active Cognition Computing research, local-first engineering, and distributed-systems work.

## Development

```sh
pnpm install
pnpm run dev
```

The production build is emitted to `dist/`. The default deployment path is `/portfolio`; override `SITE_URL` and `BASE_PATH` for a different GitHub Pages repository or a custom domain.

Content lives in `src/content`. Add a Markdown file to `research`, `projects`, or `notes` and the corresponding index and detail page will be generated automatically.
